# RSVP-Movies-SQL-Data-Analysis-for-Global-Movie-Launch-Strategy
RSVP Movies is a leading Indian film production company that plans to launch a movie for a global audience. To support data-driven decision-making, this project analyzes historical movie data from the last three years using SQL to derive insights related to movie performance, genres, directors, actors, ratings, and revenue trends.
The objective of this project is to support strategic planning for RSVP Moviesâ€™ upcoming global release by providing actionable business recommendations based on structured SQL analysis.

**Business Problem**

RSVP Movies wants to minimize risk and maximize success for its global movie release by answering key business questions, such as:

What genres perform best globally?

Which directors and actors consistently deliver successful movies?

What factors influence movie ratings and box-office performance?

How do regional and language-based trends impact audience reception?

This project uses SQL-based analytics to convert historical data into insights that guide content, casting, and release strategy decisions.

**Objectives**

Analyze historical movie performance using SQL.

Identify high-performing genres, directors, and actors.

Evaluate movie ratings and audience engagement patterns.

Perform multi-table analysis using optimized SQL queries.

Deliver insights and recommendations to support global movie planning.

**Dataset Overview**

**Domain:** Entertainment / Movie Analytics.

**Time Period:** Last 3 years.

**Data Source:** IMDb-based movie database.

**Data Model:**

Multiple normalized tables (movies, genres, ratings, people, roles, etc).

Relational structure supported by ERD.

**Key Attributes:**

Movie metadata (title, release year, language).

Financials (budget, revenue).

Ratings (IMDb score, votes).

Talent (directors, actors).

Genre classifications.

**Tools & Technologies**

SQL.

MySQL / PostgreSQL (SQL-compliant).

Relational Databases.

Joins (INNER, LEFT, RIGHT).

Subqueries.

Aggregations (COUNT, SUM, AVG).

GROUP BY, HAVING.

Window Functions (where applicable).

Business Analytics.

Data Validation & Optimization.

**Analytical Approach**

The analysis was structured into four logical segments, each answering critical business questions through SQL:

Data inspection and schema understanding using ERD.

Multi-table joins to combine movie, rating, genre, and talent data.

Aggregations and filtering for performance evaluation.

Ranking and comparison of movies, actors, and directors.

Optimized SQL query writing with meaningful aliases.

Validation of outputs using provided checkpoints.

**Key Analysis Performed**

**Genre-wise** analysis to identify globally successful categories.

**Director-wise** performance based on ratings and consistency.

**Actor analysis** to identify high-impact talent.

Movie performance trends based on **ratings and audience votes.**

**Language and regional insights** relevant for global releases.

**Key Insights**

Certain genres demonstrate consistently higher audience acceptance across regions.

Specific directors show strong performance reliability over multiple releases.

High audience votes combined with strong ratings indicate global appeal.

Strategic casting plays a significant role in movie success.

Data-backed planning reduces uncertainty in global movie launches.

**Business Recommendations**

Prioritize genres with proven global performance.

Collaborate with directors demonstrating consistent audience approval.

Consider actors with strong engagement metrics for global appeal.

Use historical ratings and votes as key decision-support metrics.

Adopt data-driven planning for content and casting strategies.
